The purpose of this game is to use the controls to move the wizard to the fruit. When the wizard touches the fruit, it magically changes it to another fruit of a kind. I used a forever loop for the keys when the game starts. I also used the forever loop on the fruit sprite to change the shape when the wizard "touches" it. The difficulties I found in this game was when I had to display the image of the fruit to another location after it was touched.

The process I went to produce this program is by starting with a variable and then printing out the story. I made desicion points which are made by raw_input. I had trouble keeping track of where or which part i'm coding, so I started making a variable for each print and desicion points. I felt great about my work. I disliked how repetitive I was.
